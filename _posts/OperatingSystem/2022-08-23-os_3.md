## **💡 ROM(Read Only Memory)**

* 컴퓨터를 구동하기 위한 기본적인 정보가 저장되어있다.
* 비휘발성 메모리이며, 정보를 영구적으로 저장한다.

------

## **📌 RAM & ROM 비교**

| RAM             | ROM              |
|:----------------|:-----------------|
| 휘발성 메모리         | 비휘발성 메모리         |
| 읽기/쓰기           | 읽기               |
| 사용자가 변경 가능      | 사용자가 변경 불가       |
| 전원이 차단되면 데이터 삭제 | 전원이 차단되어도 데이터 유지 |
| 속도가 빠름          | 속도가 느림 |
