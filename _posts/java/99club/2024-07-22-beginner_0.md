## **언어 / 문제 난이도** ##
Java / Lv.1

------

## **문제 설명** ##
자연수 n을 뒤집어 각 자리 숫자를 원소로 가지는 배열 형태로 리턴해주세요.<br/>
예를들어 n이 12345이면 [5,4,3,2,1]을 리턴합니다.

##### **제한사항**
- n은 10,000,000,000이하인 자연수입니다.

##### **입출력 예**

| n     | return      |
|:------|:------------|
| 12345 | [5,4,3,2,1] |

------

## **문제 풀이** ##
~~~java
class Solution {
    public int[] solution(long n) {
        int[] answer = new int[String.valueOf(n).length()];

        int index = 0;
        while(n > 0) {
            answer[index++] = (int) (n % 10);
            n = n / 10;
        }
        return answer;
    }
}
~~~
